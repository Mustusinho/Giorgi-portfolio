# Web Content QA Checklist (Remote / Async)

## Before publishing
- [ ] Correct page title + meta description (matches intent)
- [ ] One clear H1; headings follow order (H2/H3)
- [ ] Links: no broken links; correct destinations
- [ ] Images: compressed; descriptive alt text
- [ ] Consistent terminology + brand voice
- [ ] Accessibility basics: contrast, labels, keyboard focus
- [ ] Mobile: no overflow; buttons tappable
- [ ] Performance: large images, heavy embeds reviewed

## After publishing
- [ ] Confirm page renders correctly in production
- [ ] Confirm tracking (if applicable)
- [ ] Add change log entry (what + why)
- [ ] Monitor for 48h: errors, broken links, feedback
